* Functions

### 🔹 INTERMEDIATE

* Advanced Functions
* Object-Oriented Programming (OOP)

* Modules and Packages

* Iterators and Generators

### 🔹 ADVANCED

* Context Managers and Metaclasses

## 🚀 How to Run

1. Clone the repository:

   ```
   git clone https://github.com/kruthikadevi/practicePython.git
   ```

2. Navigate to the folder:

   ```
   cd practicePython
   ```

3. Run any Python file:

   ```
   python BASICS/01_variables_and_datatypes[BASICS].py
   python INTERMEDIATE/10_oop[INTERMEDIATE].py
   python ADVANCED/15_decorators[ADVANCED].py
   ```
